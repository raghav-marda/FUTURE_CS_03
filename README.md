# ЁЯУМ Project Overview

This repository contains the submission for **Task 03** of the **Future Interns Cyber Security Internship**.

The objective of this task was to perform practical **API Integration & DevTools Network Analysis** on the **Smart Expense Tracker** web application using **Chrome Developer Tools**.

The project focused on analyzing:

* Frontend-backend communication
* Fetch/XHR requests
* Request payloads
* JSON responses
* Session handling
* Request headers
* Network monitoring workflows

The assessment was performed through browser-side API inspection and request-response analysis.
